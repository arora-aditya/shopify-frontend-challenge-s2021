## Build steps

Confirmed compatibility with `yarn v1.22.4` and `node v12.20.0`

```sh
yarn install # Install packages
yarn start # Run locally
```

## Dependencies
- ReactJS (in TypeScript)
- Polaris
- ReactRouterDOM
- NodeSASS
